# R Analysis Code

This directory contains selected R scripts associated with the RepairDis
manuscript and DDR-state analyses.

## `manuscript_figures`

Scripts copied from the manuscript figure workspace. They generate or assemble
pan-cancer MO-DDRscore, immune remodeling, TIDE, checkpoint and DDR
gene-drug-pathway panels used in the current manuscript figures and
supplementary figures.

## `dprs_luad`

Scripts copied from the LUAD DPRS analysis workspace. They include data
processing, MO-DDRscore-associated analysis, machine-learning survival model
construction, DPRS validation and published-signature benchmarking scripts.

## Notes

These scripts retain local project assumptions and may require path updates
before rerunning. Large raw datasets and generated intermediate files are not
included in this repository.
