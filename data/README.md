# Data Tables

This directory contains small curated tables used by the RepairDis manuscript:

- `TableS1_236_curated_DDR_genes.csv`: curated DDR gene list.
- `TableS2_Final_DPRS_model_genes_and_coefficients.csv`: genes and
  coefficients retained in the final LUAD DPRS model.
- `TableS3_Published_signature_benchmark_comparison.csv`: published DDR- or
  DNA repair-related prognostic signatures used for benchmarking.

Large raw and processed omics datasets are not included in this repository.
